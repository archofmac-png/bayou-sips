# Bayou Sips

> Self-serve frozen drink machine rentals. Sip. Chill. Enjoy.

Static marketing site for Bayou Sips, a frozen drink machine rental business serving
weddings, birthdays, showers, graduations, corporate events, and backyard parties.

## Stack

Plain HTML, CSS, and vanilla JavaScript. No build step. Deployed via GitHub Pages.

## Local preview

Any static file server will do. From this directory:

```sh
python -m http.server 5173
# then open http://localhost:5173
```

## Files

- `index.html` — the page
- `styles.css` — all styles (no preprocessor)
- `script.js` — booking-form interactivity and copy-to-clipboard
- `assets/` — flyer images (menu and packages)
- `robots.txt` — crawlers allowed, including AI search agents
- `sitemap.xml` — section anchors
- `llms.txt`, `llms-full.txt` — AI-search-engine-friendly summaries
